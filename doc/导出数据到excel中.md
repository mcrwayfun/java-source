# 导出数据到excel表格中

## 1. 环境
- jdk1.8
- apache-poi-3.11

## 2. 文档
- http://poi.apache.org/index.html

## 3. 步骤
引入依赖
```
    <dependencies>
        <dependency>
            <groupId>org.apache.poi</groupId>
            <artifactId>poi</artifactId>
            <version>3.11</version>
        </dependency>
    </dependencies>
```

**测试类位于com/qingtian/下的ExcelTest**
